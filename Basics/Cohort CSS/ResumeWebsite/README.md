# 🌐 Resume Portfolio Website

Welcome to my **Resume Portfolio Website** — a clean and minimal personal webpage built using pure **HTML** and **CSS**.
This project is part of my journey toward becoming a full-stack developer, where I focus on building strong frontend fundamentals before moving into advanced frameworks.

🔗 **Live Demo:**
https://projectsnew-phi.vercel.app/CSS%20Projects/ResumeWebsite/index.html

---

## ✨ About The Project

This portfolio website is designed as a digital resume that highlights personal details, skills, and professional information in a structured and visually simple layout.

The goal behind this project was to understand how layout, typography, and styling work together to create a professional web presence without relying on frameworks.

---

## 🎯 Key Features

* Clean and minimal resume-style layout
* Structured sections for profile information
* Responsive styling using CSS
* Beginner-friendly and lightweight design
* Organized content hierarchy for better readability

---

## 🛠️ Tech Stack

* **HTML5** – Structure and semantic layout
* **CSS3** – Styling, alignment, and visual design

---

## 📚 What I Learned

While building this project, I practiced:

* Writing semantic HTML structure
* CSS positioning and layout design
* Typography and spacing for professional UI
* Building a portfolio without external libraries
* Deploying static websites using Vercel
